# 📊 Retail Sales Analysis System (Python Project)

## 🧾 Project Overview
This project is a complete Retail Sales Analysis System built using Python in a Jupyter Notebook. It analyzes the Global Superstore dataset to extract meaningful business insights such as sales performance, profit trends, customer behavior, and product analysis. The goal is to understand how data can be used to improve business decisions.

---

## 🎯 Objective
- Analyze overall sales performance  
- Understand profit trends across regions  
- Perform category-wise and segment-wise analysis  
- Identify top-performing products  
- Calculate key business KPIs  
- Extract actionable business insights

## 📊 Dataset Information
- Dataset: Global Superstore Dataset  
- Rows: ~51,290  
- Columns: 24  
- Includes sales, profit, customer, product, region, and shipping data  

---

## 🧹 Data Cleaning Process
- Removed missing values  
- Deleted duplicate records  
- Standardized column names  
- Converted date formats  
- Cleaned text data (lowercase, trimmed spaces)  
- Handled outliers using IQR method  

✔ Final output: Clean dataset ready for analysis

## 🔍 Key Insights
- Technology category generates highest sales  
- Some regions have high sales but low profit  
- Few products generate most revenue  
- Consumer segment is most valuable  
- Sales and profit are not always proportional

## 📌 Final Conclusion
The business is performing well in terms of sales, but profit optimization is required in some regions and categories. This project shows how Python can convert raw data into powerful business insights.
